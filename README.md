# 喝 

一款基于 Kotlin 开发的 Android 原生极简饮水记录 App。

## ✨ 核心特性

- **简洁**：无任何多余功能，快速记录。
- **手势与触觉反馈**：支持通过上下滑动液面动态调节单次饮水量（10ml - 300ml）。
- **高度自定义主题**：支持 **Hex 色值输入**，随心定义液体与 UI 主题色。适配 **深色模式 (Dark Mode)** 与浅色模式。
- **本地数据管理**：支持将饮水记录导出为表格文件（.csv），或从本地文件导入历史数据。


## 🚀 快速开始

请按以下步骤安装：

1. 打开本项目 GitHub 页面的 [Releases](https://github.com/your-username/drink-app/releases) 标签页。
2. 下载最新版本的 `apk` 文件。
3. 允许“安装未知来源应用”后进行安装即可。
